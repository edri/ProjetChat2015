Ceci est le répertoire final du projet Chator.

La documentation se trouve dans le dossier documentation/ et se présente sous la forme de quatre fichiers en format PDF.

Les sources de l'application sont dans le dossier src/ et un fichier Readme.txt en donne les instruction de compilation.

Les binaires de l'application sont dans le dossier bin/ et un fichier Readme.txt en donne les instruction d'exécution.
